Instructions on how to run project locally:

React Proj (Front End)
- 
1) cd into *connect-react*
    - (First time use only) 
    - run *npm install* for first time to install necessary packages
2) run *npm start*


Node API (Back End)
-
1) cd into *connect-api*
    - (First time use only) 
    - run *npm install* for first time to install necessary packages
2) run *npm start*
